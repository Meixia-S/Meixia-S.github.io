Marble Solitaire
Object Oriented Design

A fun marble solitaire game with three different board shapes to chose from.
The shapes are English, European, and Triangle. If you would like to play a
game, simply clone the repository and create a configuration. The format
should be (board shapre, , ,).
This was made with the view, model, and controller set up. The view only
deals with what the view sees (ie the user interface), while the controller
houses and processes user input, and the model handles the game logic and
functions. This keeps the code very easy to read and to edit in the future.
If you are an employer and would like futher detail feel free to contact me
since this program was not as explicit with the game design and directions.
*This was an individual project

Image Processor
Object Oriented Design

This is a image processor program that allows you load, edit, and save
images (supports: jpg, png, bmp, ppm3). There are three different ways to
use the program:

1. through the shell via text commands
2. through the terminal/scripts
3. the GUI (recommended)

For more information about the inner workings of the program please read
the README provided. The file will give a more detailed explanation about
why I designed the program in the format that I did and how to use it.
Unlike the previous project mentioned above, this assignment was more open
ended as students had more freedom over the design. The only requirements
where to make sure that all the features where working as intended. If you
would like to use the processor you can clone the respoitory. I highly
suggest that you throughly read the README file in the 'info' folder. The
program has a restrictive naming convention for saving image when using
the GUI. Let me know if you are having any issues.
*This was partially a group effort project

Country's Economy
Macroeconomics

This project focus was to research another country's macro-economy, convert
data into graphs, and analize those graphs to determine the strenghts and
weaknesses of their ecnomy. This project was done in stages during the entire
semester and at the end we needed to present our findings to the class. Due
to the fact I took the course during COVID, the presentation is a prerecorded
video. For my group we chose to reseach China. My role in this project was
the 'leader' as I consistently organized team meetings, created the graphs,
made the powerpoint, and edited the video. This experince gave me some
practice in dealing with a language barrier and working with people in
drastically different time zones.

Chain Expansion
International Business

The goal of this project is to determine weather or not a given American
chain creating a international branch in a foreign country would be an
lucrative endeavor. We had to research the different factors that would
play a role in the success of this project. Some examples are the current
trends regarding eating habits, the country's resources, infrastructure,
and the cost of building a new location as well as upkeep. The chain my
group chose was Panera and the forgien country is Brazil. By the end of the
researching phase we concluded that it would be a financally stable move
since the demographic was going towards healthier eating and the rescources
needed are present. The role I played in this project is similar to the
previous one with the only difference was having one other teammate who
would help me with research and communicating with other members.
